# Netflix-Clone
This is the link of my Netflix Clone.
https://abdulrehman9988.github.io/Netflix-Clone/

A simple Netflix clone built using HTML, CSS, and Bootstrap to replicate the UI of Netflix.

## Features

- Responsive design using Bootstrap
- Homepage layout similar to Netflix
- Navigation bar with logo and sign-in button
- Hero section with a background image and text
- Grid-based movie/show sections
- Footer with links similar to Netflix

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
